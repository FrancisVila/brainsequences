Add a citations mechanism in the description.
For the editor, typing the '#' character opens a modal requesting the title and the url of the publication.
For everyone, citations appear as links in the form "[n]" (n being an integer, ), linked to citations below the description. Each citation appears in html in the form
 <a href="link to the scientific publication concerned">[n] Citation title</a> .

Above and below the description, a "citation" toggle shows or hides the "[n]" links inside the description and the list of citations below. Use the "shut eye - open eye" pair of icons to represent the state of the citations.

In the database, the citations are linked to the step. 
If the user is logged in, rememeber the selection for show or hide citations

The default value for the citations is hide.
